# IT English Trainer

IT English Trainer is a simple web application for practicing English used in IT support, helpdesk, troubleshooting, tickets and job interviews.

The project was created with HTML, CSS and JavaScript. It runs directly in the browser and does not require a backend.

## Features

- IT support flashcards
- English translation quiz
- Typing practice
- Ticket reply simulator
- Client chat practice
- IT interview practice
- Daily challenge
- IT dictionary with categories
- Mistakes review
- Achievement system
- Points, streak and level system
- Dark and light mode
- Progress saved in the browser with localStorage
- Animated sparkle effect in quiz answers

## Technologies Used

- HTML
- CSS
- JavaScript
- localStorage

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open the `index.html` file in your browser.

No installation is required.

## How It Works
The application contains a collection of IT support phrases in English and Polish.
Users can practice by using flashcards, choosing answers in a quiz, typing translations, writing replies to support tickets and practicing interview answers.
The application saves points, correct answers, streaks, mistakes, achievements and theme preferences in the browser.

## Purpose
The goal of this project is to help beginners improve English skills for IT support and helpdesk work.
It can be useful for people who want to practice common technical phrases, user support replies and basic IT interview answers.
